Our data sources are located Here!
We used beers.csv and breweries.csv.
We created data frames of the 2 .csv files with Pandas and renamed some of the columns to names that are easier to identify. We applied a format the the "abv" column to show a percentage, used .fillna() to enter empty strings in the beer style column and a '0' in the ibu column if any data was missing.
